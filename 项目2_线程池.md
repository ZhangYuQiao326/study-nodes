## 2 多线程种类

1. IO密集型程序：io操作比较多的程序，打开文件、设备、网络操作等
   * 单核：适合并发程序
   * 多核：适合并发程序
2. CPU密集型程序：计算操作比较多的程序，深度学习框架计算等
   * 单核：不适合并发，涉及到大量的上下文切换操作，即不断往寄存器保留和读取线程信息
   * 多核：适合并发

## 4 线程同步

1. 线程互斥

   互斥锁mutex

   原子操作atomic

2. 线程通信

   条件变量

   * 生产者-消费者模型

     ```cpp
     // 生产
     int arr[]; // 资源
     condition_varable cond;
     mutex mtx;
     
     unique_lock<mutex> lock(mtx);
     while(arr.full()){
         cond.wait(lock); 
         // 生产
         cond.notify_all();
     }
     
     unique_lock<mutex> lock(mtx);
     while(arr.empty()){
         cond.wait(lock);
         // 消费
         cond.notify_all();
     }
     ```

   * wait做的事情

   * 1. 将进程状态由进行态改为等待
     2. 释放锁
